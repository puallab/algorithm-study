# algorithm-study

## 규칙
- fork로 각자 레포지토리 가져가서 진행 
- 각자 문제 풀고 자기 이름의 폴더에 풀이 올리기(Only .java file)
- 매주 코드리뷰 전 (월요일 22:00) PR 하기 
 
- 문제 출제 순서
  - 백준 기준 실버2 ~ 골드3,
  - 완전탐색, 백트래킹, DFS & BFS, 투포인터, 이분탐색, 유니온파인드, 문자열, 누적합, 
 - 토의시간(참여 가능한 사람만)
   - 매주 월요일 22시
   - 온라인 google meet 
   - 다음주 개념과 이번주 문제 풀이 공유
